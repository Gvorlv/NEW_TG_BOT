Телеграмм бот поддержки коллцентра 
=========================

1. Описание проекта
-------------------
Это телеграмм бот использующий дообученную chat GPT. 
Боту можно задавать вопросы вызывающие затруднение у оператоов коллцентра.
В папке faiss_index содержатся файлы векторизированной базы знаний.
Перед запуском бота необходимо в корневом каталоге приложения создать файлы
users.txt (для хранения логов о новых пользователях запустивших бот, 
из этих логов берется ID пользователя телеграмм для добавления в файл access.txt) из файла
access.txt берется список ID пользователей которым разрешено пользоваться ботом, айдишник
туда должен заноситься вручную администратором приложения

3. Установка
------------
1. Клонируйте репозиторий:
   git clone https://github.com/username/projectname.git
2. Перейдите в папку с проектом:
   cd NEW_TG_BOT
3. Установите зависимости:
   poetry  shell 
   poetry install 

4. Использование
----------------
Для запуска основного скрипта выполните:
   python bot8.py

5. Конфигурация
---------------
Перед запуском настройте файл `.env`, 
указав там свои параметры
TOKEN=111111111 (токен телеграмм бота)
GPT_SECRET_KEY=sk-proj11111 (токен api open ai )
